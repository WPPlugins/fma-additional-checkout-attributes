Woocommerce checkout field editor allows you to add custom fields on checkout page by drag-n-drop.

== Description ==

 
Woocommerce Checkout Manager allows you to add extra custom fields to your order page. WooCommerce Checkout fields editor plugin helps to collect additional information from your valued customers that helps you process their orders smoothly.

### Key Benefits Of WooCommerce Checkout Manager ###

* Personalize checkout pages for enhanced user experience
* Allow users to write a personal message while ordering
* Obtain more information for marketing and promotions
* Compliance with recommended user experience web design

### Key Features of WooCommerce Checkout Field Editor ###
 
**Add Woocommerce Custom Fields**

With additional fields on checkout pages, you can allow customers to be more specific about their orders, delivery dates and shipping addresses.

**Enable or Disable Fields**

Woocommerce Checkout Field Editor Plugin allows you to enable or disable any field to control the display of additional attributes on checkout pages. It empowers you to quickly manage varying fields.

**Add Different Types of Fields**

This plugin supports following types of Woocommerce Custom checkout fields;

* Text Field
* Text Area
* Select Box
* Check Box
* Radio Button

**Select Mandatory Fields**
 
You can mark the fields required or not required, so that customers shall never skip filling mandatory details.

**Configure Attributes Settings**

Customize titles of Woocommerce checkout fields to create ease for customers to read and fill the fields comfortably.

**Display Additional Attributes on checkout page**

The custom checkout fields added on billing and shipping pages can be displayed on checkout pages, so that customers can view the details they have entered.

**Enable the display of fields on order Thank you page**

The additional information provided by customers on Woocommerce checkout page can be displayed on order thank you page as well, for their records.

**ADDITIONAL FEATURES of Woocommerce Checkout Field Editor**

* Fields for Drag and Drop file upload
* You can hide or unhide fields
* Easy sorting of fields with drag up and down 
* Add labels to each field
* Field Placeholder
* Field width (Full or Half)
* Options adding for select or multi-select field

== Installation ==

1. Once the plugin is installed, activate it from the WordPress admin panel. After successful installation, navigate to Checkout Fields Attributes in backend menu. From the settings, you can set caption for the additional information section. Fields added in this section will appear separately on checkout page.

2. Now click on checkout field attributes menu, drag and drop custom fields to billing, shipping or additional information section. Configure the setting of each field accordingly.
